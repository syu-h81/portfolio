ABOUT

自己紹介

ABOUT career

Web制作会社にて、フロントエンドエンジニアとして、コーポレートサイト等の構築を主に担当。
その後、サーバーサイド、バックエンドにも関わり、技術の幅を広げて参りました。

具体的な業務経験は、Webサイトのコーディングやアニメーションなどの動的サイトの実装、WordPressへのサイト構築・カスタマイズ、プラグインの作成も得意としています。

ABOUT me

1998年生まれ、広島県尾道市出身。大学進学を気に、上京する。その後、大学を中退。 大学を中退後、独学でフロント系の言語を習得し、フロントエンドの会社にてフロントエンドエンジニアの業務を約1年経験。

今後の展望は、バックエンドの知識・技術を習得し、フルスタックエンジニアとして活躍していきたい。

SKILL

現在の技術レベル

HTML/CSS

デザインに沿ったコーディングが可能です。タグなどは、積極的に活用していき、SEOを深く意識したコーディングを提供出来ます

Sass

デザインに沿ったスタイルシートが可能です。Sassを利用することで通常よりもコードの量を大きく減らすことを目的として提供出来ます

Javascript

アニメーションを利用した動的サイトの実装が可能です。jsのライブラリやフレームワークも活用し、無駄のないプログラムを提供出来ます

JQuery

jsと同様、アニメーションを利用した動的サイトの実装が可能です。JQueryの特徴でもある、HTML/CSSの操作を簡単に操作させるようにプログラムを提供出来ます

Vue.js

アニメーションを利用した動的サイトの実装が可能です。また、コンポーネント化してパーツを制作、SPAの開発を提供出来ます

PHP

お問い合わせフォームやログイン機能の実装等、webアプリの開発が可能です。wpの構築やデータベースなどの経験があり、基本的なバックエンド側のプログラムを提供出来ます

WordPress

オリジナルサイトの組み込み、テンプレートテーマのカスタマイズが可能です。その他にも、functions機能の追加、プラグインの作成、ブログ化をしたり制作・運用も提供出来ます
